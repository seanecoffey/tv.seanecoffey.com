Why donate?
I love doing this for all of you, so you don't have to! However, the more people that use this, and the more content I make available, the cost for running this increases.
Currently, the costs run at about:
  • Generic server costs (CPU/RAM/Bandwidth for serving files to you) - $20/month
  • SSD Block Storage (Storing the files, 500GB) - $40/month
  • Additional basic routing server for self-hosted VPN - $5/month
  • Total - $65/month
You can help me out by donating anything you can to one of the options below.
Donorbox
Donorbox donations are my preferred donation method, it takes less money off the top and allows for one-off or recurring donations through Stripe.

PayPal
PayPal is obviously an old favourite but doesn't offer recurring donations unless I make myself a business etc. One-off donations via PayPal are greatly appreciated! You can send me money at the email address seanecoffey@hotmail.com or use the donate button below.


Again, there is no onus on you to donate, as I've obviously set this up by choice myself, and you didn't ask me to. However, donating allows me to create a better more complete service for you all, which I'm more than happy to do provided I have the financial resources to do so!
