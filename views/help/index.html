How does it work?

SickRage
SickRage keeps a list of TV shows we want to keep track of or get any episodes for and automatically downloads, renames and sorts the episodes.
Deluge
The Deluge web client is used for manually adding movies (these are a lot easier to deal with than TV shows).
Plex
Plex is used to serve the media files to you in a beautiful way! Plex pulls metadata and subtitles for the media, manages users, transcodes the media if required and presents it to you in a beautiful UI available across multiple devices!

This all runs with a few other behind the scenes pieces on a cloud computer instance currently based in Singapore. This server maintains high upload and download speeds, and should have the necessary hardware to maintain solid performance, but please let me know if you run into any performance issues.
